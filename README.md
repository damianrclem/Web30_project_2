# Web 3.0 Twitter Clone Project on the Blockchain

A replica of Twitter Clone application that demonstrates Minting on the blockchain.


# Frameworks used
- Alchemy
- Solidity
- Hardhat
- Sanity.io (Data storage)
- React / NextJS
- Tailwind css 3.0

# Run project locally
from the the /client folder run the following
```
yarn dev
```
